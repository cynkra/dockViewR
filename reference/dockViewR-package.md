# dockViewR: Layout Manager Widget for R and 'shiny' Apps

Provides R bindings to the 'dockview' 'JavaScript' library
<https://dockview.dev/>. Create fully customizable grid layouts (docks)
in seconds to include in interactive R reports with R Markdown or
'Quarto' or in 'shiny' apps <https://shiny.posit.co/>. In 'shiny' mode,
modify docks by dynamically adding, removing or moving panels or groups
of panels from the server function. Choose among 8 stunning themes (dark
and light), serialise the state of a dock to restore it later.
